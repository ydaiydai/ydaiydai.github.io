Beakon

"Beakon" is a 2D platform-adventure game made in Unity with a theme of Restoring the World. It was made in a team of 8 during USC MEGA Newbies & Vets Game Jam 2021. This game won Best Art amoung 16 final game submissions.

As the lead level designer and one of the programmers in this team, set up the scene and designed the overall level. Implemented the game's player mechanics and some camera/environment features.

Following the Game Jam's Theme of Restore, our team created a game where the player has to navigate through the world by moving the character and "paint" the world back to color using a "paint brush." This game was made possible by amazing works from the whole team. Everybody contributed to the design of this game. All assets used in this game were original.

I further expanded my knowledge on Unity and became more familiar with game development process. Seamless collaborations amoung all team members made this Game Jam an unforgettable experience.

Check out our itch.io page for Beakon.
The entire code-base with assets can be found on Github.

Features

The Key features of Beakon are:

  • One complex level featuring parallax effect.
  • Uniquely designed Art Theme.
  • Engaging environment reveal mechanic.
  • Dynamic animations for the main character and NPCs.
  • Original Soundtrack and sound effects.

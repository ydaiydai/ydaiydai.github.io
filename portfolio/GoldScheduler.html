Gold Scheduler

Gold Scheduler is a webapp made using HTML, Python, JavaScript under the Flask Framework with Heroku's PostgreSQL. Gold Scheduler is a webapp for UCSB students to help them plan their fututre quarter schedules.

My teamates and I worked together using the Agile Development approach. Working on this project gave us a better mindset on teamwork and greatly strengthened our collaboration skills as we assigned each other to different roles, established different objectives, and had frequent meetups and discussions. You can click here to go to the website.

Gold Scheduler is a project which I got to experience first hand a thorough development process for a webapp from the backend such as database to the frontend such as user experience, while using the formal Agile software development methodologies.

I was the UX coordinator of the team who focused on the general design and frontend framework. I worked closely with our team's backend developers and I was the Agile retrospective leader hosting a team meeting to reflect on and organize the project development.

The entire code-base with its assets can be found on Github.

Web Features

Key features of the webapp are listed below:

  • Simple User Interface that is easy to understand.
  • Gold Scheduler uses weekly calenders for the user's course planning. The student can select a subject from the top dropdown list, the relative information will be shown in the following fields so specific "Class", "Lecture", and "Section" can be selected.
  • Ability to add and drop selected class.
  • Added course is be shown on the weekly calender with its relative information marked on the corresponding time slot.
  • Different courses are shown with different colors for the user to distinguish.
  • The user can add new plans for multiple potential course schedules. A new empty calender will be initialized and the user is free to go to or delete any plan created.
  • Easy access to UCSB and Rate My Professor website.
  • Detection and notice when two courses have time conflicts.

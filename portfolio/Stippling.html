Image Stippling

This is an Image Stippling work I made which I am proud of when I was learning offline rendering. It creates a stippling image from an arbitrary image in C++. Two different mappings were implemented, one uses pure 2D random uniform seuqences and the other one uses 2D stratified sequences.

This program uses LodePNG, a .png library to obtain values of PNG images and output modified images.

Both uniform and stratified sampling algorithms use a two-step method to the image's 2D distributions. The method first sums up each column and calculates the resulting row, then sums up each row and calculates the resulting column based on the sum of rows and columns. While sampling the distribution of pixels, uniform sampling maps black dots from pure uniform random numbers; stratified sampling maps a grid like random sequence which draws a sample in each grid cell. The number of samples for the total number of pixels can be specified.

The source code for two mapping implementations including the library can be found here.
